### How to run

```
npm start
```

### What need to do before run

- `npm install`
- Add file config.json under **config folder** containing as below

```
# config/config.json

{
  "development": {
    "username": "your_username",
    "password": "your_password",
    "database": "your_db",
    "host": "your_host",
    "dialect": "postgres"
  }
}
```
